# niconico_honsha
## HowToUse
### Setup
```bash
python3 -m pip install -e .
```

### Run
#### Explode honsha
```bash
honsha_explode
```

#### Remove file, then explode honsha
```bash
honsha_rm <destination file/directory>
```
